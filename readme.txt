1. clone or download project
2. download node.js (https://nodejs.org/uk/) and install it
3. open folder with project in terminal (commad line)
4. print "npm start" and press enter
5. open browser's address line and print "localhost:9000"
6. if you need folder for prodaction, you should print "npm run buid" in terminal (commad line) and press enter. In "public" folder with be create all necessary files for prodaction.
